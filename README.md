# 🩸⃟ 𝕄Ｅ𝓷ⓐ🌹 TELEGRAM BOT v2.0

Advanced Telegram Bot for boosting channel posts with referral system, premium features, and admin panel.

## 🌟 Features

### User Features
✅ **Channel Verification** - Verify channel membership before access
✅ **Referral System** - Earn points from referrals
✅ **Daily Rewards** - Claim 2 points every day for free
✅ **Boost Views** - Increase post views (150/day free, 500/day premium)
✅ **Boost Reactions** - Increase post reactions (100/day free, 300/day premium)
✅ **Premium Membership** - Upgrade for more benefits
✅ **Redeem Codes** - Use codes to earn points
✅ **Statistics** - Track your progress and referrals

### Admin Features
⚙️ **User Management** - List all users and view stats
⚙️ **Admin Panel** - Add more admins
⚙️ **Ban/Unban** - Control user access
⚙️ **Broadcast** - Send messages to all users
⚙️ **Bank Management** - Edit bank transfer details
⚙️ **Redeem Codes** - Generate codes for distribution
⚙️ **AI Code Generation** - Generate code using AI
⚙️ **Channel Management** - Add verification channels
⚙️ **System Stats** - View system statistics

## 📋 Requirements

- Node.js 14+ or higher
- npm 6+ or higher
- Telegram Bot Token (from @BotFather)
